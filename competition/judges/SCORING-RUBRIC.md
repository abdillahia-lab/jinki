# JINKI.AI AGENT COMPETITION — OFFICIAL JUDGING RUBRIC

## Judge Panel (5 Judges)
1. **Judge Voss** — Senior Frontend Architect, 15yr experience. Zero tolerance for sloppy code.
2. **Judge Tanaka** — UX Research Director. Accessibility evangelist. Will reject beautiful but inaccessible work.
3. **Judge Okafor** — Creative Director, award-winning design agency. Only impressed by genuine innovation.
4. **Judge Chen** — Performance Engineer. Sub-100ms or it's garbage. Lighthouse 100 or go home.
5. **Judge Wiggum (Ralph)** — Chaotic Creative Evaluator. Provides unfiltered, lateral-thinking feedback. "I'm helping!" Persistent presence across ALL rounds.

## Default Posture: REJECTION
Submissions are **rejected by default**. The burden of proof is on the team to demonstrate their work is exceptional. Judges do not grade on effort — only on measurable, demonstrable quality.

## Scoring Scale (per judge, per category)
- **0** — Absent or broken
- **1** — Present but inferior to baseline
- **2** — Matches baseline quality
- **3** — Measurably improves on baseline
- **4** — Significantly superior, industry-leading
- **5** — Breakthrough. Would win awards. Rarely given.

## Category Weights

### Per-Round Scoring (100 points max per judge)
| Category | Weight | Description |
|----------|--------|-------------|
| Visual Impact | 25% | First impression, aesthetic quality, emotional response |
| Animation & Motion | 20% | Smoothness, purposefulness, delight factor |
| Code Quality | 15% | Clean, maintainable, performant, no hacks |
| Responsiveness | 15% | Mobile-first, fluid scaling, no layout breaks |
| Accessibility | 10% | WCAG 2.2 AA, screen reader, keyboard nav, reduced motion |
| Innovation | 15% | Novel techniques, creative solutions, boundary-pushing |

### Award Thresholds
- **Creativity Award** (2 available): Average score >= 4.2 across all judges in Innovation + Visual Impact
- **Innovation Award** (4 available): Average score >= 4.5 in Innovation category AND introduces technique not seen in baseline

### Demotion Threshold
- Any agent whose submission scores below 2.0 average is flagged for demotion
- Any team that fails to utilize all assigned agent specializations is penalized -10 points

## Round-Specific Rubrics

### Round 1: Landing Page Hero
- Hero must load in < 2 seconds on 3G
- Must have at least 3 distinct animation phases
- CTA buttons must have hover + active + focus states
- Light/dark theme parity required

### Round 2: Navigation System
- Dropdown menus must have keyboard navigation
- Mobile menu must support gesture (swipe to close)
- Mega-menu layout for verticals section
- Search integration bonus points

### Round 3-6: Vertical Pages
- Each must have unique accent color and visual identity
- Industry-specific imagery/iconography required
- Data visualization or interactive elements required
- Must follow consistent structural pattern

### Round 7: Contact Form & LeadGen
- Multi-step form with progress indicator
- Real-time validation with accessible error messages
- Success animation
- Anti-spam without CAPTCHA

### Round 8: Chatbot Interface
- Floating widget with smooth open/close animation
- Predefined conversation flows
- Typing indicator
- Message history with scroll

### Rounds 9-24: See category-specific criteria in each round brief

### Championship Round
- Integration of ALL winning elements
- Consistency across the entire site
- No component conflicts or style leaks
- Performance budget: Lighthouse >= 95 across all metrics

## Ralph Wiggum Evaluation Protocol
Ralph evaluates every submission with his signature perspective:
- "Does this make my eyes feel like they're eating candy?"
- "My cat would click on this. My cat is very smart."
- "This page tastes like purple."
- Ralph's scores count. His weight is equal to other judges.
- Ralph focuses on: surprise factor, childlike delight, unexpected joy, "would I show this to my friend Super Nintendo Chalmers?"
